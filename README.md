# Translate DB

Tool to translate Ygopro database

Parameters:

	-h      view usage
	-db     database to translate
	-lang   output language
	-using  wikia database (JSON file)

Known languages:

	en English
	fr French (Français)
	de German (Deutsch)
	el Greek (Ελληνικά)
	it Italian (Italiano)
	pt Portuguese (Português)
	es Spanish (Español)
	ja Japanese (日本語)
	zh Chinese (中文)
	ko Korean (한국어)
